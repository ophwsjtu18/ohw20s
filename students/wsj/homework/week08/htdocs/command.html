Parameters (Optional):
Main Command contains "move" and "house".
"move" needs at least one paramter, which indicates the direction.
"u" means up, "e" means east, and so on.
The second parameter is optional, if specified, you can determine how much distance you wanna move. 1 is the default value.
So, you can type in parameters like: "u 10", which means you'll be TPed 10 meters(blocks) above your original position.
